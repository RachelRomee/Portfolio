ConfuZr

Two player game build with HTML canvas and Javascript

    Game Rules

  • If you touch one of the blocks,
    game over!

  • When a red block passes,
    use the other key to jump

  • The black block changes gravity,
    so your block will fly up

I built this game with two guys from Codaisseur. Our goal was to make a multiplayer game, preferably with ReactJS Front-End and FeathersJS Back-end. The first day we started out in React, and there were some challenges. React doesn’t support HTML canvas by default, so we found an npm package called ‘React Canvas’, which had outdated documentation that we couldn’t figure out. Also, React support JSX instead of JavaScript, so the syntax is different. Since I never made a JavaScript game before, and we had 1 day left to finish the game, we decided to continue in JavaScript.

This project really help me to gain more understanding of JavaScript. I think that making a game is a good way to improve your coding skills, since you translate the game rules into code.
