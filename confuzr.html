ConfuZr

Two player game build with HTML canvas and Javascript

    Game Rules

  • If you touch one of the blocks,
    game over!

  • When a red block passes,
    use the other key to jump

  • The black block changes gravity,
    so your block will fly up

I built this game with two guys from Codaisseur. - group project (3) - 2 days to finish - goal: multiplayer game with feathersjs api and react frontend - challenge: react canvas & js experience - learned: I think building a game is a good way to learn code.
