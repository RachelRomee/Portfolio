To Do App

Simple to do app with a Rails API and React App.

I made this project on my own, as an assignment for Codaisseur. You can check out the code for the frontend and backend in the Github links below.

First I made the backend API in Rails, with the Rack-Cors gem to turn the Rails App into an API. The gem was easy to use and I set up the backend in under an hour. Using Rails as a backend API worked really well for me and I'm sure I'll use it in the future.

The next step was the React frontend App. I set up React with Webpack loaders (a module bundler) and Babel (to use JSX, for compatibility with older browsers). Since it was my first React App, I did have some issues with getting the data into my app. I posted a question on Stackoverflow and fixed the problem within minutes. After I got the hang of it, React was really nice to work with, using components, inline styling and states. The documentation from Facebook is not very clear though. I would like to keep working with React, so now I'm doing extra courses on React at TreeHouse, Egghead and Codecademy. Also, I want to learn more about React Native, to be able to make ios/windows/android applications. From what I read, React Native and React are very similar, so it should be easy to adopt.
